# InfoCollect

InfoCollect is a modern data collection and analytics web application built using PHP and MySQL.

## 🚀 Features

- User Registration & Login
- Secure Authentication System
- Dashboard Interface
- Data Submission Forms
- Market Sales Tracking
- Buying Trends Analysis
- Supplier Information Management
- Responsive Modern UI

## 🛠️ Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- XAMPP

## 📊 Project Purpose

This project is designed to collect and manage market-related data, including:

- Sales information
- Buying trends
- Supplier performance
- Market insights

## 🔐 Authentication

The system uses:
- PHP Sessions
- Password Hashing
- Secure Login System

## 💻 Installation

1. Clone the repository

```bash
git clone https://github.com/Mohamed-saYid-Mohamed/infocollect.git
```

2. Move the project to XAMPP htdocs folder

3. Import the database:
- Open phpMyAdmin
- Create database: `infocollect_db`
- Import `schema.sql`

4. Start Apache and MySQL in XAMPP

5. Open:

```bash
http://localhost/infocollect
```
